# 高股息央企资源股配置策略笔记

## 1. 原话引用（策略定义）

> 分红可持续且当下买入股息率必须双倍于无风险利率，roe大于等于15%，负债率50%以下，优选是央企，山海关以北国企全部否；行业门槛必须高，最好是垄断性质行业，如石油、电力等；20pe以下，2pb以下（最好1以下）目前主要集中在煤炭、电力（水电风电）、石油、航运，这四个占据6-8层仓，还有一些恒生科技、纳指100以及行业etf，保证大部分资金能吃息就行了，小仓位做做t，追追热门

---

## 2. 技术指标（可量化筛选条件）

### 2.1 核心财务指标

| 指标 | 条件 | 说明 |
|------|------|------|
| 股息率 (Dividend Yield) | ≥ 2 × 无风险利率 | 当前约 ≥ 3.5% |
| ROE | ≥ 15% | 盈利能力筛选 |
| 资产负债率 | ≤ 50% | 控制杠杆风险 |
| PE (TTM) | ≤ 20 | 估值约束 |
| PB | ≤ 2（优选 ≤1） | 资产安全边际 |

---

### 2.2 结构性指标（非直接财务）

| 维度 | 条件 |
|------|------|
| 所有制 | 央企优先 |
| 地域 | 避开北方国企（治理风险假设） |
| 行业属性 | 垄断 / 准垄断 |
| 行业门槛 | 高资本 / 牌照 / 资源壁垒 |

---

### 2.3 行业筛选池

当前策略聚焦：

- 煤炭
- 电力（火电 / 水电 / 风电）
- 石油（上游为主）
- 航运

---

### 2.4 Python筛选器示例（可复现）

```python
import pandas as pd

def filter_stocks(df, risk_free_rate=0.017):
    return df[
        (df['dividend_yield'] >= 2 * risk_free_rate) &
        (df['roe'] >= 0.15) &
        (df['debt_ratio'] <= 0.5) &
        (df['pe_ttm'] <= 20) &
        (df['pb'] <= 2)
    ]
```

## 3. 理论依据（金融与宏观逻辑）

### 3.1 股息贴现模型（Dividend Discount Model）

股票价值来源于未来现金流：

[
P = \sum \frac{D_t}{(1+r)^t}
]

核心逻辑：

* 高股息 = 高确定性现金流
* 在低利率环境下 → 估值上升

---

### 3.2 股息 vs 无风险利率（Equity Risk Premium）

定义：

[
ERP = Dividend\ Yield - Risk\ Free\ Rate
]

策略要求：

[
Dividend\ Yield \ge 2 \times Risk\ Free
]

👉 本质：

* 用股票替代债券
* 获取超额利差（carry）

---

### 3.3 行业选择逻辑

| 行业 | 理论支撑           |
| -- | -------------- |
| 煤炭 | 资源稀缺性 + 供给约束   |
| 石油 | 国家控制 + 上游 rent |
| 电力 | 公用事业 + 稳定现金流   |
| 航运 | 寡头化 + 运价周期     |

---

### 3.4 组合结构理论（Barbell Strategy）

本策略隐含结构：

* 左侧：高股息（低波动）
* 右侧：成长资产（高波动）

即：

[
Portfolio = Defensive\ Income + Growth\ Optionality
]

## 5. 讨论与分析

### 5.1 优势

#### （1）适配低利率环境

* 中国进入低利率周期
* 债券收益低 → 股息替代

---

#### （2）央企隐含安全垫

* 政策支持
* 分红稳定性较强

---

#### （3）现金流导向

* 避开“讲故事公司”
* 偏向真实盈利资产

---

### 5.2 风险

#### ⚠️ （1）周期性风险

| 行业 | 风险   |
| -- | ---- |
| 煤炭 | 煤价下跌 |
| 石油 | 油价波动 |
| 航运 | 运价崩塌 |

👉 本质：
**策略并非纯防御，而是顺周期**

---

#### ⚠️ （2）股息不可持续

* 高分红往往出现在周期顶部
* 下行周期 → 分红削减

---

#### ⚠️ （3）估值陷阱（Value Trap）

低PE ≠ 低估：

* 行业增长性低
* 资本市场给予折价是合理的

---

#### ⚠️ （4）宏观依赖

策略隐含假设：

* 利率长期低位
* 资源价格稳定

若发生：

* 利率上升
* 大宗商品下行

👉 双杀风险

---

### 5.3 可优化方向

#### （1）引入“非周期高股息资产”

* 高速公路
* 运营商
* 水务 / 燃气

👉 降低波动

---

#### （2）分离“股息”和“beta”

当前问题：

* 股息资产 = 周期资产

优化：

* 纯收息（电力/公用事业）
* 周期交易（煤/航运）

---

#### （3）动态利差管理

核心变量：

[
Spread = Dividend\ Yield - Bond\ Yield
]

需动态调整仓位

---

#### （4）市场分层

| 市场 | 功能  |
| -- | --- |
| A股 | 稳定  |
| 港股 | 高股息 |
| 美股 | 成长  |

---

## 6. 总结

该策略可以抽象为：

> 用高股息央企资源股构建“权益类债券组合”，并通过成长资产获取右侧弹性

但其本质是：

* 利率套利（carry trade）
* 周期暴露（commodity beta）

而非传统意义上的“价值投资”。
